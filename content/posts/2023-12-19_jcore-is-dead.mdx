# 前言

现在是2023年12月19日的凌晨两点半,我还有konseki在寻找的新视频的素材，我们俩打开了5ch的J-CORE総合スレ，不可避免的，我们也想了解一下现在日本的J-core爱好者的讨论，他们对J-Core的现状是什么看法。

我觉得如果你想了解J-Core，去5ch看看是很有益处。

一路看到了板块的最后，由于5ch板块满1000贴就会强制结束，每当这时候，大家就会重新开个新的板块继续讨论，但这次，大家觉得已经没有必要再开一个板块讨论了，于是持续了十多年的J-core讨论板就这样完结了。

自从TECHNORCH的J-Core文章发布已经过去了七八年了，正好Technorch的第一篇文章与第二篇的间隔也差不多是七八年，所以一直也想再写一篇关于这几年J-Core环境变化的文章。

**那么，J-Core是死了吗**

# J-Core到底是什么？

在我们讨论J-core的现状之前，我们先要搞清楚J-Core到底是个什么东西，由于这个问题一直以来都没有准确的答案，所以我们就分开来讨论

# Nerdcore Techno时期

Nerdcore Techno 是 20 世纪 90 年代初在日本出现的一种融合了当时的Hardcore与anime采样的音乐风格，比较被现在所知的大概就是高速音楽队、全日本レコード这种，这时候其实也有海外的制作人开始尝试制作这种风格，但是总体就是日本的一些Nerd用自己录像带录的采样自娱自乐的一种音乐形式

![image](/images/posts/d0b905b9d349ce5eeb754c20f0661bc76993889.png)

当然，那时候也没人会把它叫做J-core

# 本格J-Core

到90年代中后期，日本的Hardcore场景开始分裂，一边是坚持着欧洲主流风格的Record，另一边是坚持走这种Nerd风格的Record，步入了千禧年后，这种Nerdcore的制作水平也越来越高，开始有越来越多的海外爱好者注意到这种风格，他们把这种很有日本人感觉的Hardcore音乐叫做J-core

这之后，J-Core这个词传回了日本，日本的制作人对这个词也并没有一个清晰的认识，开始有制作人把自己制作的音乐叫做J-core，继续往后J-CORE这个词在日本国内也被广泛接受了，这种Anime-bootleg音乐在日本国内也越来越火

![image](/images/posts/9706037646d71e4496456ff6497f2ecc6993889.png)

渐渐的，像JAKAZID这种海外制作人也开始制作J-core音乐，大家的观念也发生了改变，虽然是外国人做的音乐，但如果是按日本人的感觉来做的话，也是J-CORE了。

![image](/images/posts/3290f2fdfbd948f250315dec84c10bf06993889.png)

# 更加广义的J-core

08年左右，关于J-core定义开始发生了第二次的分裂，原本的几乎所有J-Core制作人都受到了欧美系Hardcore音乐的影响，J-core制作人都认为J-core是欧美系Hardcore音乐的一部分。

但也开始有更多的J-core制作人完全没有受到过欧美系Hardcore的影响，单纯是喜欢J-core而开始制作Hardcore音乐。

同时像M1dy，DJ TECHNORCH这种，虽然不属于Ani-Bootleg，但也有种日本本地产的独特性，同时也与欧美Hardcore有着很大的区别，于是有一派人开始认为这些与欧美不一样，有着某种日本产的韵味的音乐也是J-core。

![image](/images/posts/6c6726e402dc2ea8a7932171c5b6d0946993889.png)

# 音乐游戏潮后

在这同时，音乐游戏也在日本不断发展，拿IIDX举例，随着玩家水平的提高，原来以House、Techno、Trance等曲风为主的曲库也开始加入越来越快（也就是Hardcore）的音乐，当然在这之前也已经有Hardcore音乐在音乐游戏出现了，但是像L.E.D\Kors K这些人与他们的音乐都是很明显的受到了欧美系Hardcore影响，甚至可以说是某种“真物”Hardcore。

在这之后，也开始出现所谓的音乐游戏特化的Hardcore音乐(很多日本爱好者都认为这种转变的开始是DJ yoshitaka与他的HARD RENAISSANCE，不过这种东西也没法考证)，开始有音乐游戏玩家开始通过音乐游戏进入到Hardcore的世界，大家也开始把这种音乐叫做J-Core音乐。

![image](/images/posts/7cae76e9d04a33354cae12f399ae19486993889.png)

...

先到这里为止吧，虽然在这之后J-Core也发生了很多的变化但恕我在这里先埋个伏笔。

到此为止，日本人的Hardcore已经与欧美的Hardcore产生了很大的分歧，文化和声音上都发生了很多的区别，这也是TECHNORCH在原文中表达的最开始的观点。

# 让人摸不着头脑的部分

那么，方向是什么

对于当时的J-Core制作人来说，J-Core到底要往哪里发展，每个人都有自己的看法，有的人觉得J-core继续往Ani-bootleg的方向发展，但也有像REDALICE这样的认为J-Core应该朝着更加“本物的Hardcore”发展，当然，事后我们都知道了谁才是对的。

# 日本人的声音不再独特

随着很多地下J-core制作人进入到了主流世界，不管是音乐游戏还是别的什么，J-Core这种声音带来的独特影响也开始作用着主流的日本音乐界。

你还记得自己已经听过多少J-POP、动画音乐、偶像音乐等等受到了很明显的影响吗？

但是属于日本人独有的声音也在这时随着逐渐消失。

2018年初，日本Hardcore圈子里非常有名的GUHROOVY SHOP闭店了，WEB的通贩也在之后慢慢关闭了

![image](/images/posts/76fc7fb2ee482058393afb71e51633586993889.jpg)

我一直认为HARDCORE SYNDROME 9发布的2015年是一个很重要的时间点，在这之后，日本人开始朝着更加本物的方向前进了。

之前花了很大的篇幅来讲原来的J-CORE音乐，现在我们可以回头来看看了：

本格的J-Core随着流媒体时代与版权化的背景下已经逐渐式微，A-Remix Nation、UNGLAND、J-core Masterz的结束，属于本格J-Core的黄金时代已经落幕很久了，随着Sharpnel等老一批J-core制作人也开始转型，本格的J-Core已经完全转入了地下。

![image](/images/posts/7bb00894158707d789270eddbf86b9e66993889.png)

更加广义的J-core在声音和文化也越来越接近主流的Hardcore，日本人开始越来越多的去尝试海外型的实践，大家可以看看现在日本Hardcore Live的内容，已经开始逐渐的转向欧美系的Hardcore演出，不管是阵容还是音乐还是形式上，同时疫情期间日本人HDM也越来越多的走向国际，YUTA IMAI和TATSUNOSHIN这样的制作人在5ch被称作日本的骄傲，带领了更多的J-Core制作人融入欧美的HDM场景。

![image](/images/posts/84b7d9841c1728484c2e06310523a6386993889.png)

“但是，像是Techcore/Futurecore，以及很多的日本制作人仍然制作着很有自己独特的音乐，日本独特的声音并没有完全消失。”可能有人会这么说。

这就是我们要讨论的另外一个问题：

# J-core不再是日本人独有的声音

HARDCORE UTOPIA 5过几天就要发售了

![image](/images/posts/1a971bd6f1b97a2c9ddbbbadb83161ed6993889.webp)

这是由很多非日本的制作人制作的日本硬核专辑，他们在声音的主体上已经和我们想象中的J-Core没有太大的区别了。

但是，他们加入了很多欧美最近流行的EDM的元素。

这种以日本的声音为基底，加入了自己的独特理解已经悄悄成为了一种潮流

Reek、Synthion、Aethral等这些Non-Japanese的J-Core制作人不用多说，还有很多的主流EDM制作人比如skybreak这种公开的表示自己的音乐受到了J-Core的影响。

另一个例子就是我们最爱的supa7onyz，在他今年那个最受欢迎魔王曲发布之前很多人都认为这首曲子是由某个日本制作人制作的，但是现在回过头发现其实曲子里有太多supa独有的声音。

![image](/images/posts/418bbbc3cf7fef2952860f524a1411996993889.png)

这就像是那些曾经受到本格J-core启发而开始制作与本格J-Core完全不一样J-Core的日本人一样，现在越来越多的海外制作人“拿走”了着这种属于日本独有的声音，创造出自己的声音。

同时，我们可以看到日本的J-Core制作人开始分为两派，一派继续制作自己曾经的风格，另外一派开始靠近欧美主流的声音，前一派就不用多说，另外一派我想举这个例子

![image](/images/posts/a3fc2cf6bf26fcde49467f9619378f396993889.png)

这首歌我其实很喜欢，但是你可以发现Massive New Krew已经完全偏离了当初自己独特的声音，从PV到音乐，完全就是欧美主流的声音。

日本独特的声音正在变得不独特，但这远远不是最严重的问题

在5ch上我发现有的人只会去TANOC TOUR，有的人只会去WR或者TDOH，这些人互相完全不能理解其他人都在干什么，并且认为自己才是正统的J-Core。

![image](/images/posts/008afdc00a2d207bd4f51e704a4717716993889.jpg)

Weekend Raver

这与二十多年前甚至十多年的J-Core场景完全不一样，那时候基本上这几种爱好者互相都有很多交际，甚至说很多人都是三者兼有。

这种变化可能是由于互联网的普及和社交媒体的兴起，人们更容易找到自己的兴趣小团体，但是这也导致了这些爱好者中信息流动的缺乏。

结果就是在这三种人看来，J-Core变得越来越无聊，越来越没有新意。

想想看吧：

TANO\*C爱好者现在都觉得TANOC的曲子能达到及格线就好了。

NERDCORE层的爱好者都觉得现在NERDCORE的情况很不妙，新兴起的Hyperflip等东西就像依托式一样污染了整个环境。

日系海外正宗Hardcore信徒......他们现在还会去听日系本格Hardcore吗？

# 文化正在萎缩

回到GUHROOVY SHOP的话题，GUHROOVY SHOP的衰弱代表了原来日本那种依靠CD养活HDM创作者的方式也逐渐在当今的日本行不懂。

当我第一次看到有人买CD就只是单纯帮某个制作人“打榜”的时候，我真的很吃惊。

原来说的“对原来在场景中的人来说是J-Core结束的一年，对现在在场景中的人来说是J-Core剧变的一年”。

现在是“对原来在场景中和现在在场景中的人来说是J-Core都已经要结束了”。

这比声音上的变化更加的严重，日本之所以可以养活这么多与主流声音不一样的人就是得益于这种消费方式。

你不需要考虑演出，也不需要考虑你的曲子能被更大众的爱好者接受，你只需要做好你自己就可以了。

但是看看现在日本，越来越多的人开始学习海外型的生存方式，这也是我认为为什么J-core开始缺少创新的原因。

另一层原因是现在绝大多数的J-Core听众，如果他们在live上听到自己没听过的曲子，他们完全不知道作何反应。

DJ当中“推荐给各位好听的音乐”这个功能也没有用了，所以大家只能一遍又一遍放大家都听出茧的曲子。

但其实换一种角度思考，对于更大范围Hardcore爱好者来说，Hardcore文化本身也是在走下坡路，这个如果要来讨论的话又是一个很大的话题，先挖一个坑。

# 现在

那么，J-Core死了吗？

我想看完我以上的想法你也会得出你自己的结论。

其实仔细一想，同人Hardcore真的可以算是一种文化奇迹，将舶来文化改为自己适用的类型，并最终影响到原来的文化。

看看国内的HDM，不也是这种类似于J-Core的发展方式吗。

“受到别的国家的Hardcore影响开始创作——开始发展出适合自己的风格——反哺世界的Hardcore文化(这个还不是很容易看到，不过我相信会有那一天的！)”

但与日本不同的是，我们一开始就是接触到三种不同的Hardcore文化

主流J-Core——NERDCORE——本格HARDCORE

在Hardcore进入到国内的最开始，大家单纯作为听众，这三种人是互有交流的，但随着开始有这三种风格的制作人出现，三种风格的爱好者之间的墙也开始建立起来了。

我当然也可以学TECHNORCH说“大家做自己喜欢的事情就好。”但是这不能解决问题，隔阂仍然会存在。

所以我想说的是:

**请不要拒绝交流，我相信如果你喜欢三种Hardcore的其中一种，你很大可能也会在另外两种找到自己喜欢的部分。**

**最重要的是做自己，而不是做一个只能呆在自己舒适区的人。**

之前有人问我为什么国内的Hardcore没有自己的声音，我想说的是如果你听了国内很多人Hardcore Producer的曲子，你会发现每个人都有自己独特的声音

![image](/images/posts/69fbb2b72e5aeaa0d4ebe03a77b630f16993889.png)

我自己觉得三种Hardcore我都很喜欢，在我的音乐里也会体现这三者不同的声音，最后形成我自己独特的声音。

我并不是一个害怕变化的人，我相信音乐也不应该只是单纯的复制其他人做过的声音，如果各位能找到自己独特的声音就最好了。

希望各位能找到自己独特的声音。
