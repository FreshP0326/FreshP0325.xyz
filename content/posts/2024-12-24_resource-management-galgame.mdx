# 如何管理自己的数字与生活资源？(Galgame篇)

# 前言

平安夜快乐！一年没有更新这个系列了，最近终于带来了Galgame组曲的第二章，也正好借此机会来聊聊ACGN中的“G”——Galgame。

怎么玩Galgame呢？笔者作为一名接触Galgame十余年的玩家，积累了一些心得体会，在这里分享给各位。

今年可能是Galgame十几年来的最低谷了，让更多人领略到文字冒险游戏的魅力。

# 硬件

理论上，你只需要一台Windows电脑就能游玩99.9%的Galgame了。

即使是一些在主机端独占的游戏，或者是一些老PC98上的游戏的话，你也可以通过模拟器解决绝大部分的问题了，（应该是几乎全部，因为我没想出来有任何一个Galgame只上了PS5没上NS的）这之后会提到。

如果你需要模拟近年来的主机，或者是需要有超分的需求，那么你可能就需要一台性能较好的电脑。

如果你有串流的需求，那么你可能还需要一个投影仪或电视或平板，我个人是接受不了在前两种如此大屏的状态下玩黄油的，如果使用平板的话，最好为了你的眼睛和颈椎考虑买一个支架，淘宝上找个二三十块能用的就好了。

# 新闻

我并不习惯去追新游戏的资讯，对于绝大部分情况我认为你只需要看Galgame批评: [https://space.bilibili.com/2072586344/](https://space.bilibili.com/2072586344/)的每周总结就可以了。

但我特别不喜欢看视频版的，一般我会在各大网站或者是微信公众号上看文字版的.

你也可以去各大Galgame论坛吹水或者是寻找资讯，但不像我其他的爱好，我个人是不喜欢和其他人交流Galgame相关的资讯了，而且对于很多人来说，这些论坛门槛比较高，大部分也不再活跃，故不推荐除非你需要很多很冷门的资源。

如果你日语好推荐你上Galgame花园: [https://galgame.aoba-e.info/](https://galgame.aoba-e.info/)，我认为应该是日系文字游戏新闻最全的地方了。

还有CnGal资讯站: [www.cngal.org](https://www.cngal.org)，收集了国产Galgame的相关信息，如果你不怎么关注国G，推荐点开尝试一下，一定会让你眼前一亮。

# 数据库

* VNDB: [https://vndb.org/](https://vndb.org/)

  VNDB是全欧美（可能可以去掉这个欧美了）最大最完整的视觉小说数据库

  功能和下面介绍的几个数据库相比不仅一个没少，还可以找到各个游戏的购买以及本地化补丁

  关于VNDB的介绍和使用说明，可以看看月幕上的这篇文章：[https://www.ymgal.games/co/article/171565455570370560](https://www.ymgal.games/co/article/171565455570370560)
* 批评空间: [https://erogamescape.dyndns.org/](https://erogamescape.dyndns.org/)

  如果放在两三年之前，相较于VNDB我更会推荐批评空间，但是这几年VNDB的信息已经非常完善了，如果你只是为了找信息那么VNDB已经完全够用了，但是在评论、TAG的分类的精细程度、以及周边情报上，还是批评空间信息更为丰富。

  关于怎么上批评空间，找站长要白名单的方式不太适合国内用户，推荐找一找反代站点。

  如果你要写评价，但你不会英语和日语的话，那你可以使用Bangumi。

# 模拟器与虚拟机

大部分情况下你的Windows10/11 PC已经能够运行99.9%的galgame，当然也有一些并未登录PC平台的galgame你需要通过模拟器进行游玩。

幸运的是，目前为止99%未登录PC的Galgame都有办法在你的PC上游玩，并且大部分模拟器都支持通过Playnite进行管理。

以下分享一些常用的模拟器：

* PSP（如圣诞之吻、侦探歌剧）：我一般用PPSSPP: [https://www.ppsspp.org/](https://www.ppsspp.org/)，虽然有PC版但大多数情况我都直接用手机模拟PSP玩这些游戏了
* NS（如月姬R、日本一近几年的作品（其实一般会上PS4PS5的都会上NS））：虽然最近一票NS模拟都被老任橄榄了，但单论文字游戏来说即使是Yuzu也能运行大部分发售的Galgame，所以并没有什么特殊的问题
* 3DS/NDS：鉴于本人玩过的DS恋爱游戏只有爱相随，而且新爱相随也已经有了比较完整的汉化补丁，需要游玩可以直接使用Citra进行游玩，
* PC模拟（包括Windows XP以及之前的Windows系统、DOS以及PC98等）：可以看看Galgame 汉化补档计划: [https://kindhearted-belief-c76.notion.site/Galgame-34e83ad5794a4eb990b217b588f40961](https://kindhearted-belief-c76.notion.site/Galgame-34e83ad5794a4eb990b217b588f40961)上相关章节的介绍，注意windows安装老版本时请安装日文系统，如果是MS-DOS和PC98上的游戏也可以看看Galgame 汉化补档 on DOS UI: [https://kindhearted-belief-c76.notion.site/Galgame-on-DOS-UI-d71870cb86ab46448e4de5a57e199873](https://kindhearted-belief-c76.notion.site/Galgame-on-DOS-UI-d71870cb86ab46448e4de5a57e199873)
* XBOX系独占：Ever17重置明年发售，要不你还要玩什么！
* PS系独占：并没有玩过，有兴趣可以自行研究

如何将模拟器加入Playnite请看官方文档: [https://api.playnite.link/docs/manual/features/emulationSupport/emulationSupportOverview.html](https://api.playnite.link/docs/manual/features/emulationSupport/emulationSupportOverview.html)

# 游戏时间统计

正常情况下，Playnite的游戏时间统计已经可以使用了，但是我经常会把游戏放在后台干活，也会有经常把游戏挂在后台就睡觉了的情况，Playnite并不像Steam那样可以统计你的挂机时长。

所以目前我统计游戏时间用的是ManicTime: [https://www.manictime.com/](https://www.manictime.com/),他是一个软件使用时长的统计软件，相比之下它不仅可以只计算你激活在前台的软件的使用时间，如果你一段时间不操作也会为你标记上**离开**，这就使得统计游戏时间变得可能。

# 转区

Locale Emulator: [https://xupefei.github.io/Locale-Emulator/](https://xupefei.github.io/Locale-Emulator/)就不多介绍了，主要讲一下如何Playnite启动游戏时自动进行转区。

第一种：

1. 在Playnite你需要转区的游戏上右键，编辑游戏——指令
2. 将路径修改为LEProc.exe
3. 将参数修改为-runas "LEConfig.xml里的Guid" "需要启动的游戏目录"

第二种：

1. 下载LEShortcutCreator: [https://github.com/Svintooo/LEShortcutCreator](https://github.com/Svintooo/LEShortcutCreator)，启动
2. 将你需要转区启动的EXE文件放进去，生成快捷方式
3. 通过快捷方式启动

个人目前采用第二种

# 远程控制

如果你需要远程使用手机或者平板玩玩Galgame，那么我推荐局域网使用Sunshine: [https://github.com/LizardByte/Sunshine](https://github.com/LizardByte/Sunshine)+Moonlight: [https://github.com/moonlight-stream](https://github.com/moonlight-stream)的解决方式，非局域网使用Parsec: [https://parsec.app/](https://parsec.app/)。

如果使用iPad游玩，下载一个ParsecVDisplay: [https://github.com/nomi-san/parsec-vdd](https://github.com/nomi-san/parsec-vdd)，虚拟一个你平板分辨率的屏幕，游玩4：3游戏的时候有奇效（记住Sunshine要修改到你虚拟的显示器上）

远程模拟键鼠直接手机上下载个Remote Mouse，或者Github上也有很多平替的软件。

如果需要购买实体可以看看这篇专栏：[https://www.bilibili.com/opus/976877727255625747?spm_id_from=333.999.0.0](https://www.bilibili.com/opus/976877727255625747?spm_id_from=333.999.0.0)
